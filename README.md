### Requirements:
* Composer
* Git
* Make
* PHP ^8.0

### Setup:
```sh
$ git clone https://github.com/algins/php-oop-patterns.git
$ cd php-oop-patterns
$ make install
```

### Run tests:
```sh
$ make test
```

### Run linters:
```sh
$ make lint
```
